Features

Built for developers
who ship worldwide

Every tool you need to manage Xcode string catalogs, wrapped in a native app that feels right at home on Apple platforms.

🃏

Swipe Review

Tinder-style card interface. Swipe right to approve, left to skip. Source and target side by side on every card. Review an entire language in minutes.

❌Skip
🌍Hola
✅Approve
📊

Language Dashboard

Progress bars, missing counts, needs-review badges. Know exactly where every language stands at a glance.

🔗

GitHub Integration

Browse repos, download .xcstrings files, edit translations, and create pull requests — without leaving the app.

📄

Native .xcstrings

Full Xcode string catalog support. Edit translations inline, update states, modify comments. Fully compatible.

🎨

5 Color Themes

Midnight, Ocean, Forest, Sunset, Nebula. Dark mode that feels personal.

🔒

Privacy First

Your data stays on device. GitHub tokens stored securely in Keychain. No analytics, no tracking.
