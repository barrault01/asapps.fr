// Features

Everything you need to ship faster

Built for developers who manage multiple repositories. No more context switching between browser tabs.

Unified Dashboard

See pull requests from all your watched repositories in one glance. No more jumping between tabs.

One-Click Merge

Merge, squash, or rebase directly from the app. Choose your strategy, customize the commit, and ship.

Auto Refresh

Stay up to date without lifting a finger. PRs refresh automatically every 15 minutes in the background.

Menu Bar Access

Quick-glance PR count and list right from the macOS menu bar. Always just a click away.

CI Skip Support

Append [ci skip] to commit messages with a single toggle. Skip unnecessary builds on trivial merges.

Delightful Details

Haptic feedback on actions, animated backgrounds, and confetti on successful merges. Shipping code should feel good.
