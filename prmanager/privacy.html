Information Collection and Use

PRManager connects to the GitHub API on your behalf using your authentication credentials (OAuth token or Personal Access Token). This connection is made directly from your device to GitHub's servers.

We do not collect, store, or transmit any of your data to our own servers. All data — including your GitHub token, repository information, and pull request data — is stored locally on your device and in your iCloud account (via SwiftData/CloudKit) if enabled.

Specifically:

  • Your GitHub authentication token is stored securely in the device Keychain.
  • Your list of watched repositories is stored locally using SwiftData.
  • Pull request data is fetched on-demand from the GitHub API and is not persisted beyond the current session.
  • No analytics, tracking, or telemetry data is collected.
  • No third-party advertising or analytics SDKs are included in the app.

Third-Party Services

PRManager communicates exclusively with the GitHub API to provide its functionality. Your use of GitHub is subject to GitHub's Privacy Statement.

No other third-party services are used by the app.

Log Data

We do not collect log data. In the event of an app crash, Apple's standard crash reporting may collect anonymous diagnostic information if you have opted into sharing diagnostics with app developers through your device settings.

Cookies

PRManager is a native app and does not use cookies.

Security

Your GitHub token is stored in the system Keychain, which provides hardware-backed encryption. All network communication with the GitHub API uses HTTPS. We do not have access to your credentials or data at any point.
